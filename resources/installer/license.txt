JARVIS — голосовой ассистент для Windows

Программа с открытым исходным кодом (GPL-3.0).
Репозиторий: https://github.com/120spartacus120-hash/jarvis

Для работы JARVIS использует микрофон и сеть (распознавание речи, проверка обновлений).
Антивирус может показать предупреждение у неподписанных сборок — это не означает вирус.

Рекомендуется устанавливать только файл *-setup.exe с официальной страницы Releases на GitHub.

Нажимая «Принимаю», вы соглашаетесь с условиями лицензии GPL-3.0.
