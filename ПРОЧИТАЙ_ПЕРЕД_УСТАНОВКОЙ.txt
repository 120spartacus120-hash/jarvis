JARVIS — установка для пользователей
====================================

Скачивайте только с GitHub Releases:
  https://github.com/120spartacus120-hash/jarvis/releases/latest

Файл: JARVIS_1.0.0_x64-setup.exe  (или новее по версии)
Это нормальный мастер установки (NSIS), не архив и не VBS.

Если Windows пишет «неизвестный издатель» или «вирус»
------------------------------------------------------
Программа пока без платной подписи Microsoft — так бывает у новых exe.
Это НЕ значит, что JARVIS вирус.

Что делать:
  1. Скачать только с официального Releases (ссылка выше).
  2. Подробнее → Выполнить в любом случае (Run anyway).
  3. Не ставить старые MSI *0.0.3* и не копировать папку target\release.
