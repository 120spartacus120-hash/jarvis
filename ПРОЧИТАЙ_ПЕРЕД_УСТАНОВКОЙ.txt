JARVIS — как поставить НОВУЮ версию (1.0.0)
============================================

Почему после установки «всё как раньше»?
--------------------------------------
1. Это НЕ кэш настроек. Настройки лежат в %AppData%\com.priler.jarvis —
   они не меняют вид главного экрана.

2. Чаще всего ставят СТАРЫЙ установщик. В папке bundle\msi\ лежат три файла:
   - jarvis-app_0.0.3_x64_en-US.msi   ← СТАРЫЙ
   - JARVIS_0.0.3_x64_en-US.msi       ← СТАРЫЙ
   - JARVIS_1.0.0_x64_en-US.msi       ← НУЖЕН ЭТОТ

3. Или запускают не из «Пуск», а старый ярлык / ► ЗАПУСК JARVIS.vbs
   из папки разработки — там старый exe без установщика.

4. В шапке v0.1.0 = старая сборка. В 1.0.0 должно быть v1.0.0.

Что сделать сейчас
------------------
1. Параметры Windows → Приложения → удалить все JARVIS / jarvis-app.
2. Установить ТОЛЬКО с рабочего стола:
      JARVIS_1.0.0_Installer.msi
   (или JARVIS_1.0.0_Setup.exe)
3. Запускать из меню «Пуск» → JARVIS (не старый ярлык).
4. Настройки → Основные: сверху должен быть «Автозапуск».

Папка проекта (единственная рабочая):
   C:\Users\Asus\Desktop\EXE\Jarvis\jarvis-master

Пересобрать установщик: ► СОБРАТЬ УСТАНОВЩИК.bat
